Tag Form (<from>)

Tag form merupakan tag yang digunakan saat kita akan membuat sebuah form dalam HTML. 
            

                <!DOCTYPE html>
                <html lang="en">
                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head >
                <body >
                    <form name="" method="get" action="" target="">
                    </form >
                </body >
                </html>
Dalam tag form kita dapat menggunakan atribut name, atribut action, dan atribut method dengan value get dan post. Saat kita menggunakan method get maka data/nilai akan ditampilkan pada URL, lalu ditampung oleh action sedangkan method post maka data atau nilai langsung ke action untuk ditampung, tanpa menampilkan pada URL. Selain itu juga, kita dapat menggunakan atribut target pada form kita.

Type text merupakan value yang membuat pengguna dapat memasukan input berupa text dalam form di website kita.

    Saat kita akan menggunakan tag input, kita dapat menggunakan beberapa atribut yang dapat memodifikasi input yang akan diterima. Atribut yang dapat kita gunakan adalah

    Name : Untuk menuliskan id dari sebuah bagian.
    Value : Untuk menuliskan isi dari input text secara otomatis
    Readonly : Untuk membuat input dari value tidak dapat dirubah oleh pengguna
    Disabled : Untuk membuat input pada form tidak dapat diakses oleh pengguna
    Maxlength : Untuk membatasi jumlah input yang masuk pada tag input
    Required : Untuk membuat sebuah input harus diisi oleh pengguna
    Placeholder : Untuk memberikan deskripsi singkat pada kolom input. Saat kita menggunakan atribut ini, kita tidak dapat menggunakan atribut value agar placeholder dapat terlihat.

    PHP (Hypertext Preprocessor) merupakan Bahasa server side scripting yang dapat digunakan oleh kita untuk membuat website kita menjadi lebih responsive dan kinerja yang dilakukan menjadi lebih bagus. Untuk menggunakan php, kita dapat menginstal xampp dahulu.Xampp merupakan sebuah aplikasi yang berfungsi sebagai server local/localhost yang dapat digunakan untuk membuka file PHP kita. Untuk menginstal xampp, kita dapat menginstalnya melalui google atau mengikuti panduan pada video di bawah :

    Pembuatan Form pada PHP

    Untuk membuat form pada file php, kita dapat membuatnya seperti dalam pembuatan form di fie HTML. Pada pembuatan form, kita tidak memerlukan tag php (<?php   ?>) karena pada pembuatan form, kita tidak menggunakan elemen PHP. Untuk penulisan pembuatan form, kita dapat langsung menggunakan tag form lalu menggunakan atribut nama, atribut action, dan atribut method dengan value get atau post. Setelah itu, kita dapat menggunakan tag input dengan atribut type yang kita inginkan dan atribut name dan value sesuai dengan keinginan kita.

    Untuk contoh pembuatan form pada file php dapat dilihat pada gambar di bawah :

    Halaman 1

                <html lang="en">
                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head >
                <body >
                    <form action="hasil.php" method="GET">
                    Nama: <input type="text" name="nama">

                    <input type="submit" value="Kirim" name="kirim">

                </body >

                <?php
                    $nama1=$_GET('nama');

List
 Dalam HTML, terdapat 2 jenis list.   Jenis list tersebut adalah

a. Unorder List (<ul>)

Unorder list merupakan list yang tidak memiliki urutan/tidak terpaku dengan susunan. Untuk unorder list, kita dapat menggunakan tag ul (<ul>) lalu untuk membuat list (<li>, kita dapat menggunakan tag list. Pada unorder list, kita dapat menggunakan atribut type dengan value disc, square, circle karena mereka tidak terpaku dengan urutan.

Berikut merupakan contoh dari penggunaan unorder list :

Type Disc

            

                <!DOCTYPE html>
                <html lang="en">
                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head >
                <body >
                    <ul type="disc">
                        <li>Kalimat 1</li>
                        <li>Kalimat 2</li>
                        <li>Kalimat 3</li>
                    </ul>
                </body >
                </html>

Tips :   Untuk membuat unorder list secara cepat, kita dapat menggunakan rumus “ul>li*(jumlah list yang diperlukan)” sehingga saat dienter, list akan berjumlah sesuai dengan angka yang dikali dengan list.

Untuk contoh penggunaan rumus cepat dapat dilihat pada codingan di bawah : 
           

                <!DOCTYPE html>

                    <meta charset="UTF-8">

                    <meta name="viewport" content="width=device-width, initial-scale=1.0">

                </head >
                <body >
        
                ul>li*5
                
                </body >
                </html>

    <li></li>

  </ul>

Tips :   Untuk menampilkan data didalam list secara cepat, Kita dapat menggunakan rumus “ul>li{data $}*(banyak data yang kita butuhkan)” sehingga saat dienter, jumlah data yang keluar akan sesuai dengan angka yang dikali pada list tersebut.

Untuk contoh penggunaan rumus cepat dapat dilihat pada codingan di bawah : 
           

                <!DOCTYPE html>

                    <title>Document</title>

                ul>li{Data ke $}*5
                
                </body >
                </html>

b. Order List (<ol>)

Order list merupakan list yang memiliki urutan/terpaku dengan susunan. Untuk orderlist, kita dapat menggunakan tag ol dan untuk membuat list, kita dapat menggunakan tag list. Pada order list, kita dapat menggunakan atribut type dengan value 1, i, I, a, dan A karena mereka terpaku dengan urutan/susunan.

Berikut merupakan beberapa contoh dari order list :

Description List (<dl>)

Description list merupakan list yang digunakan saat kita akan mendeskripsikan suatu hal. Untuk description list, kita dapat menggunakan tag dl (<dl>) lalu kita gunakan tag dt (<dt>) untuk menuliskan kata apa yang akan di definisi, kita dapat mengetikan tag dd (<dd>) untuk menuliskan deskripsi dari kata tersebut. Untuk contoh penggunaan tag dl dapat dilihat pada contoh di bawah: 
            

                    <!DOCTYPE html>
                    <html lang="en">
                    <head >
                        <meta charset="UTF-8">
                        <meta http-equiv="X-UA-Compatible" content="IE=edge">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Document</title>
                    </head >
                    <body >
                    <dl>
                        <dt>Team Wangy<dt>
                        <dd>Team Wangy merupakan...<dd>
                    </dl>
                    </body >
                    </html>

  Hasil :

 Team Wangy
 Team Wangy merupakan...

Tips :   Untuk memperindah dt, kita dapat menggunakan text formatting agar tulisan menjadi lebih nyaman untuk dibaca.

Table (<table>)

Tag table merupakan tag yang digunakan saat kita ingin membuat table pada file HTML kita. Untuk penggunaan tag table, kita dapat menuliskan tag table lalu dalam tag table, kita dapat menuliskan tag tr(<tr>). Tag tr merupakan tag yang berfungsi sebagai baris pada sebuah tabel.  Setelah tag tr, kita ketikan tag td (<td>). Tag td merupakan tag yang berfungsi sebagai kolom pada sebuah tabel.Untuk membuat batas dari kolom terlihat, kita dapat menggunakan atribut border sehingga table dapat dilihat oleh kita.

Untuk penulisan tag table dapat kita lihat pada gambar di bawah : 
            

                <!DOCTYPE html>
                <html lang="en">
                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head >
                <body >
        
                    <table>
                        <tr>
                            <td>Nama Kelompok:</td>
                            <td>Wangy</td>
                        </tr>
        
                        <tr>
                            <td>Jumlah Anggota:</td>
                            <td>2</td>
                        </tr>
                    </table>
        
                </body >
                </html>

  Hasil :

Nama Kelompok: Wangy
Jumlah Anggota: 2

Selain itu juga, kita dapat memperikan judul pada table dengan menggunakan tag caption dibawah tag tabel. 
            

                <!DOCTYPE html>

                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">

                        <tr>

Tips :   Untuk membuat tabel dengan baris sebanyak yang kita mau, kita dapat menggunakan rumus “tabel> (tr>td*(jumlah kolom yang diperlukan))*(jumlah baris yang diinginkan)” sehingga saat dienter, tabel akan otomatis terbuat dengan jumlah baris dan kolom sesuai keinginan kita.

Untuk contoh penggunaan rumus cepat dapat dilihat pada gambar di bawah : 
            

                <!DOCTYPE html>
                <html lang="en">
                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head >
                <body class="satab1" >
                    table>(tr>(td*2))*2
                </body >
                </html>

  Hasil :

Tips :   Untuk menampilkan data didalam tabel secara otomatis, Kita dapat menggunakan rumus “tabel>(tr>td {data ke $}*(jumlah kolom yang diperlukan))*(jumlah baris yang diinginkan)”“ sehingga saat dienter, jumlah data yang keluar akan sesuai dengan jumlah baris dan kolom yang telah dimasukan.

Untuk contoh penggunaan rumus cepat dapat dilihat pada codingan di bawah 
            

                <!DOCTYPE html>
                <html lang="en">

Atribut dan value pada tabel

Untuk membuat tabel /cell pada tabel menjadi lebih menarik kita dapat menggunakan atribut dan value yang memperindah tabel/cell pada tabel kita. Atribut dan value yang dapat kita gunakan adalah
  • bg color : Untuk mewarnai tabel/cell
  • bordercolor : Untuk merubah warna border tabel
  • width : Untuk merubah ukuran lebar tabel
  • height : Untuk merubah tinggi tabel
  • align: Untuk merubah posisi tabel pada browser.
  • cellspacing : Untuk memperbesar jarak antar cell
  • cellpadding : Untuk memperbesar ukuran cell
  • valign : Untuk merubah ketinggian tulisan pada tabel

Colspan

Colspan merupakan sebuah atribut yang digunakan untuk memperbesar kolom pada sebuah cell. Jika kita memasukan colspan pada sebuah kolom maka cell tersebut akan memperbesar sebuah cell sesuai dengan value dari atribut tersebut. Sebagai contoh, jika kita memasukan atribut colspan sebesar 3 maka cell tersebut akan memperbesar kolom sebuah cell sebanyak 3.

Untuk contoh penggunaan colspan, kita dapat melihat codingan di bawah : 
            

                <!DOCTYPE html>
                <html lang="en">
                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head >
                <body >
                <table>
                        <tr>
                            <td rowspan="3">Data ke 1</td>
                            <!--<td></td>-->
                            <!--<td></td>-->
                        </tr>
                        <tr>
                            <td>Data ke 1</td>
                            <td>Data ke 2</td>
                            <td>Data ke 3</td>
                        </tr>
                        <tr>
                            <td>Data ke 1</td>
                            <td>Data ke 2</td>
                            <td>Data ke 3</td>
                        </tr>
                </table>
                </body >
                </html>

  Hasil :

Data ke 1
Data ke 1 Data ke 2 Data ke 3
Data ke 1 Data ke 2 Data ke 3

Tips :   Untuk membuat tabel sesuai dengan ukuran yang telah ditentukan, kita dapat membuat cell yang tercolspan sebagai comment sehingga cell tersebut tidak muncul di halaman website kita dan ukuran tabel sesuai dengan ukuran awal.

Rowspan

Rowspan merupakan sebuah atribut yang digunakan untuk memperbesar baris pada sebuah cell. Jika kita memasukan rowspan pada sebuah kolom maka cell tersebut akan memperbesar sesuai dengan value dari atribut tersebut. Sebagai contoh, jika kita memasukan atribut rowspan sebesar 3 maka cell tersebut akan memperbesar baris sebuah cell sebanyak 3. 
            

                <!DOCTYPE html>
                <html lang="en">
                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head >
                <body >
                <table>
                        <tr>
                            <td colspan="3">Data ke 1</td>
                            <td>Data ke 2</td>
                            <td>Data ke 3</td>
                        </tr>
                        <tr>
                            <!--<td></td>-->
                            <td>Data ke 2</td>
                            <td>Data ke 3</td>
                        </tr>
                        <tr>
                            <!--<td></td>-->
                            <td>Data ke 2</td>
                            <td>Data ke 3</td>
                        </tr>
                </table>
                </body >
                </html>

  Hasil :

Data ke 1 Data ke 2 Data ke 3
Data ke 2 Data ke 3
Data ke 2 Data ke 3

Tips :   Untuk membuat tabel sesuai dengan ukuran yang telah ditentukan, kita dapat membuat cell yang terrowspan sebagai comment sehingga cell tersebut tidak muncul di halaman website kita dan ukuran tabel sesuai dengan ukuran awal.

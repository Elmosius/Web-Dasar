Apa itu CSS ?

CSS adalah singkatan dari cascading style sheets, yaitu bahasa yang digunakan untuk menata/mengedit file HTML yang kita miliki. Saat kita menggunakan, CSS kita dapat langsung menata beberapa file HTML secara sekaligus. Dengan CSS, kita bisa mengatur jenis font, warna tulisan, dan latar belakang halaman. Dengan kata lain, CSS berfungsi untuk memperindah file HTML kita. Untuk syntax CSS, kita memerlukan selector dan declaration. Selector merupakan elemen HTML yang ingin kita edit/diberi style. Penanda selector dapat berupa langsung elemen HTML, id, ataupun class.

Declaration merupakan elemen-elemen yang mengedit selector. Setiap declaration dibuka dan ditutup oleh kurung kurawal ({----}). Selain itu juga, setiap declaration berisi property dan value dari property tersebut. Saat menggunakan declaration, kita dapat menggunakan tanda titik koma (;) sebagai pemisah antar declaration. Untuk membuat file CSS, kita dapat membuat file di visual studio code dengan berakhiran .css (Contoh: style.css). Terdapat 3 cara untuk menggunakan file css dalam file HTML kita. Cara-cara tersebut adalah :

  • External style sheet

    • Pada penggunaan file css secara external style sheet, kita menggunakan 2 file yaitu file HTML dan file css sehingga kita harus menghubungkan file css dengan file HTML kita agar css dapat bekerja pada file HTML kita. Untuk menghubungkan file css dengan file HTML, kita dapat menggunakan tag link lalu gunakan atribut rel dengan value stylesheet kemudian gunakan atribut href dan masukan nama file css kita. Setelah itu, baru kita masukan selector dan declaration kedalam file css kita.

    Berikut merupakan cara penggunaan CSS Eksternal :

                
    
                <!DOCTYPE html>
                <html lang="en">
                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                </head >
                <body >
                    <link rel=”stylesheet” href=(nama file css).css">
                </body >
                </html>
  • Internal style sheet

    • Pada penggunaan file css secara internal style sheet, kita membuat file css dan file HTML dalam satu file saja. Untuk menggunakan file css secara internal, kita dapat memasukan tag style pada bagian head HTML kita kemudian masukan selector dan declaration yang kita pilih ke dalam tag style.

    Berikut merupakan cara penggunaan CSS Internal :

                
    
                <!DOCTYPE html>
                <html lang="en">
                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>

                    <style >  
                        body{
                            background-color:blue;
                        }    
                    </style > 
                </head >
                <body >
                   
                </body >
                </html>
  • Inline style

    • Pada penggunaan file css secara inline, kita menggunakan file css pada elemen HTML yang ingin kita edit. Untuk melakukan edit, kita dapat membuat atribut style kemudian masukan selector dan declaration yang kita inginkan.

    Berikut merupakan cara penggunaan CSS Inline :

                
    
                <!DOCTYPE html>
                <html lang="en">
                <head >
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>

                    <style >  
                        <body bgcolor="blue">   
                    </style > 
                </head >
                <body >
    Jika kita menggunakan ketiga cara penggunaan file CSS diatas secara bersamaan, CSS secara inline yang menjadi prioritas dari komputer. Hal tersebut disebabkan komputer mendeteksi CSS mana yang terdekat dengan elemen. Oleh karena itu, CSS secara inline diprioritaskan saat menggunakan tiga cara diatas secara bersamaan. Untuk penggunaan CSS secara eksternal, kita dapat menghubungkan selector dengan CSS kita. Selector yang dapat kita gunakan adalah elemen HTML, id, dan class.

    Untuk menghubungkan elemen HTML dengan CSS eksternal, kita dapat menuliskan elemen yang terdapat pada file HTML kita ke halaman CSS kita. 
                
    
                body{
                    background-color:red;
                }
                h1{
                    font-size:20px;
                }
    Untuk menghubungkan selector id dengan CSS external, kita dapat menggunakan tanda hashtag/pagar(#) lalu menuliskan id dari sebuah elemen HTML. Saat menggunakan selector id, kita tidak boleh mengawalinya dengan angka. 
                
    
                .judul1{
                    font-size:60px;
                }
                .penjelasan1{
                    font-size:20px;
                }
    Untuk menghubungkan selector class dengan CSS external, kira dapat menggunakan tanda titik(.) lalu menuliskan class dari elemen HTML tersebut. 
                
    
                .judul2{
                    color:white;
                }
                .penjelasan2{
                    text-align:center;
                }
    Selain elemen HTML, id dan class, terdapat juga Selector Universal. Selector universal merupakan selector yang otomatis memlihi semua elemen HTML yang ada pada halaman. Selector universal digunakan dengan hanya mengetikan tanda bintang (*) saja . 
                
    
                {
                    text-align:center;
                    font-size:20px;
                }

    Komentar

    Pemberian komentar pada CSS cukup berbeda dengan pemberian komentar pada HTML. Untuk memberikan komentar pada CSS, kita dapat menuliskannya dengan tanda garis mirng dan bintang(/*) lalu isi komentar kemudian ditutup dengan tanda bintang dan garis miring(*/).

    CSS Box Model

    Untuk pembuatan CSS, kita harus mengetahui CSS box model. Model tersebut adalah Elemen yang kita masukan dalam HTML disebut sebagai content. Area yang ada disekitar content disebut padding. Padding juga dapat berfungsi sebagai jarak antara content dengan border. Pembatas padding suatu content disebut border. Bagian luar dari border disebut margin. Margin juga dapat digunakan sebagai jarak antar suatu border dengan border lainnya.


    CSS memiliki banyak sekali property yang dapat digunakan untuk mengedit halaman HTML kita. Oleh karena itu, kita dapat melihat property CSS melalui link dibawah ataupun melalui W3School. Properti CSS pada W3School Berikut merupakan beberapa contoh dari property CSS (link) :

    CSS Background CSS Border CSS Margin CSS Padding CSS Height CSS Width CSS Font CSS List CSS Table Layouts

                Body{
                    Background-color:#ffffff;
                    Background-image:url(“img tree.png);
                    Background-repeat: no-repeat;
                    Background-position:right top;
                }

    Menjadi

                
    
                Body{

                }
